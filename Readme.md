# First Start  redis in your mac:

-   `brew services start redis`
-   `redis-cli`
-  `ping`

### Hit the url to see results from your localhost

`http://localhost:3001/publish`



#   Install Commands

-  `npm init -y `
-  `npm i redis `
-  `npm i express `
